Plastic Deposit Refund System

A full-stack web application that incentivizes plastic recycling by providing digital refunds, eco-points, and real-time environmental impact tracking.

 Problem Statement
Plastic waste is one of the biggest environmental challenges today.  
There is no transparent, motivating, and digital system that rewards citizens for returning plastic waste.


 Solution
We built a *Plastic Deposit Refund System* that allows users to:
- Scan plastic products
- Return plastic bottles via smart bins
- Receive instant wallet refunds
- Earn eco-points
- Track CO₂ saved
- Allow government/admin authorities to monitor recycling impact

 Environmental Impact
- Each plastic bottle recycled saves *~50g of CO₂*
- Impact is tracked in real time

 Conclusion
This system demonstrates how technology can encourage sustainable behaviour through incentives, transparency, and data-driven governance.
